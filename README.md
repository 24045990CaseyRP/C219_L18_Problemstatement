# Republic Polytechnic: School of Infocomm Course Enrolment Portal

## Objective
A custom-built React application designed for the School of Infocomm at Republic Polytechnic. This portal allows prospective students to explore various diplomas, view module details through dynamic nested routing, and register their interest in specific courses.

## Features
* **Diploma Explorer**: Browse all diplomas offered by the School of Infocomm.
* **Nested Routing**: Deep-link access to module details via `/diplomas/:diplomaId/:moduleId`.
* **Registration System**: A dedicated form to collect student interest with automatic confirmation messages.
* **Responsive Design**: A professional, academic UI that works across mobile and desktop devices.
* **Custom UI**: Creative application of Republic Polytechnic's branding, typography, and layout.

## Technical Requirements Met
1.  **Dynamic Data Rendering**: Modules and diplomas are fetched and rendered dynamically.
2.  **React Router v6**: Implementation of nested routes and index routes.
3.  **Component Extraction**: Clean code organization through modular component design.
4.  **Version Control**: Regularly committed history on GitHub.

## Installation & Setup
1. Clone this repository to your local machine.
2. Run `npm install` to install project dependencies.
3. Run `npm start` to launch the portal in your browser.
